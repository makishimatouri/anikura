# Changelog

本项目从 `v0.1.0` 开始按正式版本留档。所有上线版本都应有 Git tag，并在本文件记录用户可见变化。

## v0.1.1 - 2026-07-09

仓库规范化整理。

- 重写 README，使仓库首页更像正式项目
- 新增架构、部署、版本管理和维护文档
- 新增 CI 检查流程和 PR 模板
- 移除会制造无意义提交的 weekly backup workflow
- 补齐 ESLint 9 配置，并修复现有 lint error
- 清理 `.env.local.example`，避免出现疑似真实密钥的占位符
- 保持线上功能逻辑不变

## v0.1.0 - 2026-07-09

基线版本归档。

- 上线 Anikura CN 网站基础功能
- 支持全国活动浏览、筛选、详情页和 QQ 加群
- 支持 AniROX 活动专区、联系页、关于页和友情链接占位
- 支持管理员登录、活动录入、编辑、审核和通知
- 支持首页精选轮播
- 支持用户注册、每日签到、积分、优惠券兑换和门票抽奖入口
- 支持 Supabase 后端、Vercel 部署、Cloudflare 自定义域名

## 历史说明

仓库里保留过早期 tag `v0.3.0`。该 tag 属于开发阶段误命名，不作为正式版本线。正式版本线以 `v0.1.0` 为起点。
